## Project 1: Quantitative
### Pantone movements in color
This initial phase of the project helped me to assess the data structure, rethink my inial UI, and evaluate the feasibiity of subsets like "period", "media" and "year". I experimented with Tableau to more easily represent the data with the given data structure, and learned a lot about the Cooper's API specificities, limits, and methods.
